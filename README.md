# Loan-Default-Prediction-System

## Dataset
The dataset used in this project is sourced from [Kaggle - Lending Club Loan Data](https://www.kaggle.com/datasets/wordsforthewise/lending-club). It contains historical loan data with features such as annual income, credit score, and loan status.

### Preprocessing Steps
1. Removed rows with missing values.
2. Encoded categorical variables using one-hot encoding.
3. Scaled numerical features using StandardScaler.
4. Split the data into training (80%) and testing (20%) sets.
